# @stackloom/canvas-ui

Reusable canvas-focused React UI components built on @stackloom/ui.

## Install

npm i @stackloom/canvas-ui

## Peer dependencies

- react, react-dom (v18 or v19)
- tailwindcss (v4)

## Usage

import { CanvasPanel } from "@stackloom/canvas-ui"

export function Demo() {
return <CanvasPanel />
}

## Exports

- "@stackloom/canvas-ui/_" -> "./src/_.tsx"
