# @stackloom/builder-ui

Reusable builder-focused React UI components built on @stackloom/ui.

## Install

npm i @stackloom/builder-ui

## Peer dependencies

- react, react-dom (v18 or v19)
- tailwindcss (v4)

## Usage

import { BuilderPanel } from "@stackloom/builder-ui/panel"

export function Demo() {
return <BuilderPanel />
}
