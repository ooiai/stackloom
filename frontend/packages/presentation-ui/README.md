# @stackloom/presentation-ui

Reusable presentation-focused React UI components built on @stackloom/ui.

## Install

npm i @stackloom/presentation-ui

## Peer dependencies

- react, react-dom (v18 or v19)
- tailwindcss (v4)

## Usage

import { PresentationPanel } from "@stackloom/presentation-ui"

export function Demo() {
return <PresentationPanel />
}
